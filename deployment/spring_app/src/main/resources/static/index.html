Prédiction ECG

Interface de test pour le modèle déployé. Le signal attendu contient 96 valeurs.

Depuis un signal texte

Coller les valeurs du signal, séparées par des virgules, des espaces ou des retours à la ligne.

À utiliser pour tester rapidement une seule série.

Depuis un fichier

Formats acceptés : .txt, .csv ou .tsv.

Une ligne complète du dataset est acceptée si la première colonne contient le label.

Aucune prédiction lancée.
